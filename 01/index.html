RO.ME "3 DREAMS OF BLACK"

The RO.ME demo, created by Google in 2010, was a 3D city built with WebGL and HTML5 that users could explore in their web browsers. It showcased the creative potential of modern web technologies and inspired developers to use these tools for immersive online experiences.

Technology: HTML5, WebGL, JavaScript, CSS3, Web Audio API

Comparison: ! No Fishing Whereas RO.ME is an experience and technical showcase of a new technology, No Fishing is an informative, advanced use of the same technology.
